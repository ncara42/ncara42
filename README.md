## About me

Full-stack developer with a systems background from **42 School**. I build web and mobile products end-to-end — fintech platforms, scrapers, APIs, iOS apps, and developer tooling.

I care about architecture that holds up in production, security that doesn't get in the way of UX, and shipping things that actually work.
